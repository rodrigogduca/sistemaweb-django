Bem-vindo ao Sistema de Membros

Este projeto é para gerenciar membros e suas tarefas.
Você pode cadastrar, editar, remover e consultar membros, além de atribuir tarefas a cada um.

  • ✔ Cadastro, edição e remoção de membros
  • ✔ Cadastro, edição e remoção de tarefas para cada membro
  • ✔ Visualização das tarefas de cada membro
  • ✔ Interface moderna e responsiva
